iSMS - Single-Molecule FRET Microscopy Software
Version 2.01

HOW TO RUN
Run iSMS.m in MATLAB: 
 - Open iSMS.m and press the green run button in the Editor toolbar

GET STARTED
For documentation, quick guides and updates go to: http://isms.au.dk
Test data and a demo session can be found on the website.

REQUIREMENTS
This version has been tested positive in R2013b, R2014a, R2014b and R2015a.
You need MATLAB R2013b or newer with the following toolboxes installed:

 1) Image Processing Toolbox
 2) Optimization Toolbox
 3) Statistics Toolbox

In order to use parallel computing you additionally need the Parallel Computing Toolbox.

You need administrator rights at the installation directory path.
